# Mister AI Windows Tool

A Windows desktop tool created by **Mister AI**.

## Features
- Extracts emails from huge files
- Removes duplicates automatically
- Saves extracted emails
- Email counter
- Neon cyber UI
- Sound click effects
- Smooth fade-in transitions

## Run
pip install -r requirements.txt  
python src/main.py

## Build EXE
pyinstaller --onefile --windowed src/main.py
